# Wireless Signal VR project

## Working with Unity
- This is developed with Unity 6.
- Developing Unity version is "6000.0.26f1".  The Unity version should match to run and test.
- If the exact version of Unity can't be installed, try with a slightly higher version.

## Working with git
- After git clone, change to the 'develop' branch to see all the working updates to build and run to test.
- If a person plans to work on it, it’s recommended to check out a new branch from the ‘develop’ branch with the person’s name or task (ex: dev-john), and work on that branch.  Once the work portion is done, merge the changes from (ex: dev-john) to the “develop” branch.  Keep the “develop” branch only for the merging branch and avoid changing it directly.
